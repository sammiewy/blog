Hexo + github 搭建自己的博客

发表于 2016-04-09   |   分类于 技术

1. 配置环境

安装node，在node官网上下载node，进行安装

安装Git, 下载Git进行安装

2. 安装HEXO

安装hexo

进入你的blog目录，初始化配置

启动本地服务, 再浏览器查看http://localhost:4000/

3. 部署到github上

生成静态资源

生成的静态资源在public目录下，把public的全部资源拷贝出来，push到gh-pages分支上
